## Front-end installation ##

1. **Install dependencies**

	If you donâ€™t already have npm installed, head over to the [Node.js website](https://nodejs.org/en/) and download the relevant copy of Node.js for your system. The npm program is included with the install of Node.js.

	Once you have npm installed, open up Terminal (or Command Prompt) and enter the following command:

    `npm install`		

2. **Install Gulp-cli**

> The Gulp command line interface.

Install this globally and you'll have access to the gulp command anywhere on your system.

	`npm install -g gulp-cli`

3. **Run project**

	`npm run start`
